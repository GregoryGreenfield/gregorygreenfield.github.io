Projects

Automatic String Tuner

Designed, fabricated and programmed an electromechanical automatic string tuner for my final year university dissertation.

The PCB microcontroller picks up the frequency of the string using a zero-cross detection (ZCD) algorithm, and tightens or slackens the tension in the string to adjust the frequency to 46.25Hz, known in musical notation as F#1/Gb1.

Rote Language Learner

Developed a rote learning tool to help me learn natural language vocabulary that tests me on specific words multiple times. It works by connecting to a SQL database populated with words to learn.

I learnt Git for implementing version control, Docker for containerisation, SQL for relational database creation and querying and aspects of C# code such as try and catch statements for exception handling.

Are LINACs In Use?

Created a .NET WinForm that determines and displays which of the ten LINACs are currently being used.

I learnt about GUI creation and how to create a front-end that ensureds the user can easily garner the relevant information. information visible at a glance.

Christmas Card Generator

Created a Christmas card generator that prints a Christmas message and Manelbrot inspired trees and snowflakes, which are generated using recursive functions.

The character curves were created using fibonacci sequences, whilst the non 90° angles (such as those in W and Z were determined using triggernometry.

Calibration Files Webscraper

Wrote a script that scrapes Elekta's website for the calibration files for our 10 linear accelerators using Selenium WebDriver module.

The format of the calibration files were standardised using a PowerShell script, afterwhich the data was collated using Excel, where the data was used to help diagnose system faults, reducing down time.
